# Algorithm-Study
✔알고리즘 문제 풀이 스터디 

## 스터디 규칙
- 각 스터디 원은 매주 일요일 밤 12시까지 문제 풀이한 내용을 자신의 remote 저장소에 push한 후 master에 PR을 날린다. (master <- fork master)
- PR을 날린 후, 단톡방에 다음 주 목표를 올린다. 
- 알고리즘 문제는 프로그래머스 코딩테스트 연습 문제로 한다. 
- 언어에는 제한이 없으며 자신이 연습하고 싶은 언어로 한다.
